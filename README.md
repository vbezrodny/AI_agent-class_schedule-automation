# 🎓 Schedule Parser Agent

**Автоматический парсер расписания занятий из PDF в календарь (ICS, JSON)**

Агент для автоматического распознавания, парсинга и конвертации расписания занятий из отсканированных PDF-документов в структурированные JSON-файлы и календари (ICS) для интеграции с Google Calendar, Apple Calendar, Outlook и другими календарными приложениями.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| **OCR распознавание** | Автоматическое распознавание текста из отсканированных PDF через Mistral AI OCR |
| **Парсинг таблиц** | Извлечение расписания из сложных таблиц с поддержкой объединенных ячеек |
| **Чередование занятий** | Поддержка формата `Предмет1 // Предмет2` (числитель/знаменатель) |
| **Разделение на подгруппы** | Автоматическое разделение занятий для `п/г 1` и `п/г 2` |
| **Элективные дисциплины** | Отдельное расписание для физкультуры и элективных курсов |
| **Генерация календаря** | Экспорт в ICS (iCalendar) для Google/Apple/Outlook |
| **Множественные потоки** | Обработка нескольких расписаний из одного PDF |
| **История обработки** | Отслеживание всех обработанных файлов |
| **Автоматическая организация** | Сортировка результатов по профилю и направлению |

## 🛠 Технологии

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| **OCR** | Mistral AI OCR API | Распознавание текста из PDF |
| **Язык** | Python 3.10+ | Основной язык разработки |
| **HTTP клиент** | httpx | Запросы к API |
| **Форматы** | JSON, ICS, Markdown | Выходные форматы |

## 📦 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/vbezrodny/AI_agent-class_schedule-automation.git
cd AI_agent-class_schedule-automation
```

### 2. Установка зависимостей

> Рекомендуется создать виртуальное окружение ( venv ) перед установкой зависимостей

```bash
pip install -r requirements.txt
```

---

### 3. Настройка API ключа

Создайте файл `.env` в корне проекта ( `src` ) и вставьте свой API–ключ:

```env
MISTRAL_API_KEY=<your_api_key_here>
```

Получить API ключ можно на [Mistral AI Platform](https://console.mistral.ai/home).

## 🚀 Использование

### Быстрый старт

1. Поместите PDF–файлы в директорию `materials/pdf/`
2. Запустите агента:

```bash
python agent.py
```

3. Выберите расписание из интерактивного списка:

```text
============================================================
📚 ДОСТУПНЫЕ РАСПИСАНИЯ
============================================================
 1. 🆕 Programmnaya inzheneriya-20-02-26.pdf
     📅 15.04.2026 14:30 | 📦 3.10 MB
 2. ✅ Biznes-informatika-20-02-26.pdf
     📅 14.04.2026 10:15 | 📦 2.45 MB
============================================================

📌 Выберите расписание:
   • Введите номер файла (1, 2, 3...)
   • Или введите название файла
   • Или введите 'all' для обработки всех новых
   • Или 'history' для просмотра истории
   • Или 'q' для выхода

👉 Ваш выбор: 1
```

4. Дождитесь завершения обработки (обычно 15-20 секунд)
5. По умолчанию результаты будут сохранены в `materials/processed/`

> Примечание* — используйте VPN, чтобы взаимодействовать с API Mistral

## 📄 Форматы выходных данных

### 1. Schedule JSON (*_schedule.json)

Структурированное расписание с информацией о чередовании:

```json
{
  "metadata": {
    "group": "09.03.04 Программная инженерия",
    "profile": "Программное обеспечение компьютерных систем",
    "academic_year": "2025-2026",
    "period": "02.02.2026-08.06.2026"
  },
  "schedule": [
    {
      "day": "ПН",
      "lesson": "1",
      "subject1": "Компьютерная графика, п/г 1, ЭОиДОТ",
      "subject2": "Компьютерная графика, п/г 2, ЭОиДОТ",
      "alternating": true,
      "alternation_details": {
        "numerator": {
          "subject1": "Компьютерная графика, п/г 1, ЭОиДОТ",
          "subject2": null
        },
        "denominator": {
          "subject1": null,
          "subject2": "Компьютерная графика, п/г 2, ЭОиДОТ"
        }
      }
    }
  ]
}
```

### 2. Calendar JSON (*_calendar.json)

События календаря с временными метками:

```json
{
  "metadata": {...},
  "calendar": {
    "total_weeks": 18,
    "total_events": 156,
    "events": [
      {
        "summary": "Компьютерная графика, п/г 1, ЭОиДОТ",
        "location": "ЭОиДОТ",
        "start": "2026-04-16T08:30:00",
        "end": "2026-04-16T09:50:00",
        "week_type": "numerator",
        "is_alternating": true
      }
    ]
  }
}
```

### 3. ICS файл (*_calendar.ics)

Стандартный iCalendar формат для импорта в любые календарные приложения:

```text
BEGIN:VCALENDAR
VERSION:2.0
BEGIN:VEVENT
UID:20260416-1-ПН-12345@schedule
DTSTART:20260416T083000
DTEND:20260416T095000
SUMMARY:Компьютерная графика, п/г 1, ЭОиДОТ
LOCATION:ЭОиДОТ
END:VEVENT
END:VCALENDAR
```

## 🎯 Особенности обработки

### Формат чередования (числитель/знаменатель)

| Формат | Что означает | Результат |
|--------|--------------|-----------|
| `Предмет` | Нет чередования | Каждую неделю |
| `Предмет1 // Предмет2` | Чередование | Числитель: Предмет1, Знаменатель: Предмет2 |
| `// Предмет` | Чередование | Числитель: нет, Знаменатель: Предмет |
| `Предмет //` | Чередование | Числитель: Предмет, Знаменатель: нет |

### Разделение на подгруппы

- п/г 1 — занятия для первой подгруппы (столбец 3)
- п/г 2 — занятия для второй подгруппы (столбец 4)

### Элективные дисциплины

- Автоматически определяются по ключевым словам:
- Элективные дисциплины
- физической культуре
- физкультура
- спорт

Для них используется отдельное расписание времени (можно настроить).

## 🔧 Устранение неполадок

### Проблема: Ошибка SSL при подключении к Mistral AI

**Решение:** Если вы используете VPN, попробуйте выполнить операцию снова.

> Примечание* — устранение данного недочета в разработке...

### Проблема: Таблицы не распознаются

**Решение:** Убедитесь, что PDF имеет четкое качество. Рекомендуемое разрешение: 300 DPI.

### Проблема: Неверное время пар

**Решение:** Отредактируйте словарь `lesson_times` в `schedule_parser.py` под ваше расписание.

### Проблема: Файл не обрабатывается

**Решение:** Проверьте:
1. PDF файл находится в `materials/pdf/`
2. API ключ корректный и активный
3. Интернет-соединение стабильно

### 📊 Статистика обработки

| Параметр | Значение |
|----------|----------|
| Время обработки 1 PDF | 15-20 секунд |
| Поддерживаемые форматы | PDF (скан/электронный) |
| Выходные форматы | JSON, ICS, Markdown |
| Максимальное количество таблиц | Не ограничено |
| Поддержка чередования | Да (числитель/знаменатель) |
| Поддержка подгрупп | Да (п/г 1, п/г 2) |
